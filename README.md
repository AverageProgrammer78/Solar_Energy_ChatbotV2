# SolarBot – Your Solar Energy Assistant

Welcome to **SolarBot**, your AI-powered assistant for all things solar energy — from understanding how solar panels work, to calculating savings, scheduling appointments and answering your renewable-energy questions.

Live demo: [https://averageprogrammer78.github.io/Solar_Energy_Chatbot_V2/](https://averageprogrammer78.github.io/Solar_Energy_Chatbot_V2/)

---

## Features

- **Chat interface**: ask questions in natural language about solar energy, panels, payback, ROI, etc.  
- **Savings calculator**: estimate how much you could save by switching to solar power.  
- **Weather & solar-irradiance data**: get a sense of local solar potential (if implemented).  
- **User authentication**: simple sign-in to manage chat history and export/delete conversations.  
  > Demo credentials: `demo` / `demo123`  
- **Chat export & management**: export conversation sessions, delete chat history, log out.  
- **Appointment scheduling (optional)**: capture leads or schedule a consultation for solar installation.  

---

## Technology Stack

- Front-end: HTML, CSS, JavaScript  
- Chat UI: Custom or existing library (depending on implementation)  
- Authentication: Simple demo auth (for production, replace with real backend)  
- Calculator logic: JavaScript functions computing savings based on user input  
- Deployment: Hosted on GitHub Pages (`gh-pages`) under repository `Solar_Energy_Chatbot_V2`  
- Optional APIs: Weather/solar-irradiance API (to power “Solar Weather” feature)
